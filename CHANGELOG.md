# 更新记录

## 1.0.7

- 新增 Obsidian 绑定与推送功能
- 支持将 Markdown、TXT、HTML 输出复制到 Obsidian Vault
- 优化本地模型选择和错误提示

## 1.0.6

- 新增 HTML 演示 PPT 输出
- 生成 `slides.md` 的同时自动生成 `slides.html`
- HTML 演示页支持浏览器翻页和打印

## 1.0.5

- 优化 PyQt6 主界面布局
- 优化处理失败弹窗，支持复制错误信息
- 优化 Ollama 本地模型读取
- 支持从本地目录选择 faster-whisper 模型

## 1.0.0

- 初始版本
- 支持视频导入、音频提取、Whisper 转写、Ollama 内容生成
- 支持 Markdown 知识文章、小红书文案、公众号稿、学习总结、标题库导出
